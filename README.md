##  Sistema de Customização e Loja Virtual (Roblox)

Projeto desenvolvido no Roblox Studio utilizando Lua, com foco em lógica de programação e criação de interfaces interativas.

###  Funcionalidades
- Sistema de seleção e visualização de skins
- Interface interativa com botões e menus
- Lógica de ativar e desativar itens
- Sistema de compra individual ou completa
- Organização de estados e eventos

###  Tecnologias Utilizadas
- Lua
- Roblox Studio

###  Objetivo do Projeto
Desenvolver um sistema funcional e organizado, aplicando conceitos de lógica de programação, usabilidade e estruturação de código.
